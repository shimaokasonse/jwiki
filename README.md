## jwiki

日本語ウィキペディアの全ページタイトルと、リダイレクト情報をまとめています。
いつでも最新版を作れるように、生成スクリプトも用意しました。

リダイレクトは、例えば「スマフォ」や「SmartPhone」などの語を「スマートフォン」に統一させて検索できるようにするものです。辞書は以下の様な感じになります：
